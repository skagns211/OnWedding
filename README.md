## What service is it?

### OnWedding
결혼을 준비 과정에서 생기는 고민, 또는 결혼 정보 등을 자유롭게 공유하며  
결혼 준비의 어려움과 부담을 덜어주고 서로 힘이 되어주는 커뮤니티입니다.
